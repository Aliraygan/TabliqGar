# نصب ربات
```sh
cd $HOME
git clone https://github.com/Aliraygan/tabchi.git
cd TabLiqGar
chmod +x install
chmod +x start
./install
./start
# سپس شماره و کد را ارسال کنید :)
# اجرای اتولانچ:
cd tabchi
chmod 777 screen
chmod +x install
./install
screen ./screen
```
